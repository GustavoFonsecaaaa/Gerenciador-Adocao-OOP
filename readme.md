# 🐾 Gerenciador de Adoção (OOP)

Uma aplicação de consola desenvolvida em C# para gerir adoções de cães. Este projeto tem como foco principal a aplicação prática de conceitos fundamentais de Programação Orientada a Objetos (POO) e estruturação de dados.

## 🚀 O Projeto
Este mini-sistema permite o registo via terminal de adotantes e relaciona-os aos seus respetivos animais de estimação. Ao final da execução, a aplicação processa os dados armazenados na memória e gera um relatório detalhado das adoções realizadas. 

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** C#
* **Framework:** .NET 8.0
* **Ambiente:** Console Application

## 🧠 Conceitos Aplicados
Durante o desenvolvimento deste projeto, foram consolidados os seguintes fundamentos do backend:
* **Classes e Instanciação:** Estruturação independente das entidades `cachorro` e `adotante`.
* **Composição de Objetos:** Relacionamento entre classes, onde a classe adotante detém uma instância do animal adotado.
* **Manipulação de Coleções:** Utilização de Arrays unidimensionais para alocar múltiplos registos em memória.
* **Laços de Repetição:** Uso de `for` para a rotina de preenchimento de dados e `foreach` para a leitura e exibição do relatório final.
* **I/O no Terminal:** Interação dinâmica com o utilizador utilizando métodos nativos da classe `Console`.
